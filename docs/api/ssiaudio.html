ssi_filter_PreEmphasis
  PreEmphasis  FILTER
 
           This component performs pre- and de-emphasis of speech signals using a 1st order difference equation: y(t) = x(t) - k*x(t-1) (de-emphasis: y(t) = x(t) + k*x(t-1)).
 
             +  de   BOOL   1   'false'   De-emphasis instead of pre-emphasis (i.e. y[n] = x[n] + k*x[n-1]) 
             +  k   FLOAT   1   '9.700000e-01'   The pre-emphasis coefficient k in y[n] = x[n] - k*x[n-1].

ssi_feature_AudioIntensity
  AudioIntensity  FEATURE
 
           Computes Intensity and/or Loudness (narrow band approximation)
 
             +  intensity   BOOL   1   'true'   enables the output of intensity I (mean of squared input values multiplied by a Hamming window) 
             +  loudness   BOOL   1   'false'   enables the output of loudness L : L = (I/I0)^0.3 ; I0 = 0.000001 (for sample values normalised to the range -1..1)
